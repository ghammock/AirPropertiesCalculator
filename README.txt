Air Properties Calculator
Author: Gary Hammock, PE  (2013)

================================================================================
                                DESCRIPTION
================================================================================

This Windows(R) application can calculate the thermodynamic and transport
properties of equilibrium air for pressures between 10 Pa and 10 MPa and for
temperatures between 0 and 30,000 K.

The program accepts inputs and generate outputs in standard SI units.

================================================================================
                                   USAGE
================================================================================

1.) Double click the executable file to launch the application.
2.) Enter the static pressure of the thermodynamic state in the Pressure field.
3.) Select either Temperature or Enthalpy as the second thermodynamic property
    used to define the state.  Enter the static temperature or enthalpy into
    the field.
4.) Click the "Calculate" button.  This will activate the "Write to File"
    button which allows you to save the state information to a plain-text file.
5.) When finished, click the "Exit" button (or the standard Windows close [X]
    button to exit the application.

5a.) The "About" button displays the licensing content that is also found in
     the accompanying "license.txt" file.  This window also contains a button
     labeled "References" that displays the references used in the calculation
     of the air properties (the same references as those found below).
